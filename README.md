# To-Do List Application (Python Project)

A simple command-line based To-Do List application built using Python.

This project helps users manage daily tasks by allowing them to:
- Add tasks
- View tasks
- Delete tasks

It also demonstrates basic Python programming concepts and user interaction through terminal input.

---

## 📌 Project Overview

The application provides a menu-driven interface where users can:

### Features:
- Add multiple tasks
- View all saved tasks
- Delete tasks using task number
- Handle invalid inputs gracefully
- Prevent empty task entries

The program runs continuously until the user chooses to exit.

---

## 🛠 Concepts Used

- Python Lists
- Loops (`while`)
- Conditional Statements
- Exception Handling (`try-except`)
- User Input Handling
- Menu-Driven Programs

---

## ▶️ How to Run

1. Install Python
2. Download the project file
3. Run the program using terminal:

```bash
python todo_list.py
```
